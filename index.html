Projects

Live Docs

LiveDocs is a web application that enables real-time document collaboration, allowing users to create, edit, and share documents seamlessly. The platform includes features like version control, live updates, and a commenting system with mentions notifications, ensuring efficient communication among team members. When a user is mentioned in a comment, they receive an instant notification, enhancing collaboration and workflow. Designed for teams of all sizes, LiveDocs ensures a smooth, intuitive experience for productive document management.

Zentry

Zentry is a modern and interactive frontend project inspired by the Zentry website. Built using React.js, GSAP, and Tailwind CSS, this project demonstrates a blend of sleek design, smooth animations, and responsive layouts. The user-friendly interface is designed to deliver a seamless experience across desktop and mobile devices. By employing clean and modular code, the project ensures easy scalability and maintainability. Key features include GSAP-powered transitions, fully responsive design for various screen sizes, and engaging UI elements, showcasing a strong focus on both functionality and aesthetics.

Nike Landing Page

A visually captivating and modern landing page designed for Nike, built with React.js and styled using Tailwind CSS. This project highlights a sleek, responsive design optimized for seamless performance across desktop, tablet, and mobile devices. With interactive UI elements and smooth animations, the landing page delivers an immersive and engaging user experience. Its clean codebase ensures maintainability and scalability, making it a showcase of both style and functionality.
